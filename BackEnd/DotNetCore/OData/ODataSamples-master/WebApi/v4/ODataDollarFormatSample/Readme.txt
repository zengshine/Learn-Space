ODataDollarFormatSample
-----------------------------------------------------------------------------

This sample shows how to use $format in Web API for OData v4.0. The sample covers:

1) Using $format with a MIME content type.
2) Using $format with a MIME content type with parameters.
3) Using $format with an alias for the format.
4) Using $format to override the value set in the accept header.

This sample is provided as part of the OData WebApi sample repository at
https://github.com/OData/ODataSamples
